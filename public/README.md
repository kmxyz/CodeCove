# Public Assets Directory

Place your image files in this directory to use them in your application.

## Required Files

- `hero.jpg` - The image for the elderly care hero section (showing elderly couple in park with bench)

To use the images, simply reference them in your components like this:

```jsx
<img src="/hero.jpg" alt="Description of image" />
```

Note that files in this directory are served at the root path, so you should reference them with a forward slash.
